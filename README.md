# dotFiles
## Locations:

Place all the folders into ~/.config **EXCEPT** .xinitrc file.


==========================================================================================
  
**.xinitrc:**
Place the the home dir (~/)
  
Don't forget to change **xrandr** args
  
==========================================================================================
**kitty.conf:**
  
Place: ~/.config/kitty/
 
==========================================================================================
